# Dogs

Dogs are the focal point of IWDR. When you input a dog into IWDR it becomes a part of
the global data set used to calculate Estimated Breeding Values on a variety of health
and behavior characteristics. The more dogs input into our database the more accurate
the calculations become.

## The dog model

The dog model contains all the pertinent information on a given dog within IWDR.

- **API Table**: MyDogs_2
- **Available Actions**: [create](#create-a-dog), [list](#list-your-owned-dogs), update
